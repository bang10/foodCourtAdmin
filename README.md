# 학식, 푸드코트 예약 관리자

## 개발기간
2024.10.16~
## 개발환경
- IDE: WebStorm
- FrameWork: Vue3, Quasar
- TypeScript
- node 18.20.4
- npm 10.7.0

## 설명
사용자의 상태, 허가된 음식점 등을 관리할 수 있고 구매내역을 확인할 수 있습니다.
협력사의 상품 등록, 수정, 삭제 등을 할 수 있고, 협력사의 상품을 구입한 사용자 등을 알 수 있습니다.

## 실행 방법
```
yarn
# or
npm install

quasar dev
```
